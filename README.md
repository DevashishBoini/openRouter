# Spring Boot Backend

A simple Spring Boot REST API application with PostgreSQL database.

## Prerequisites

- Java 17+
- IntelliJ IDEA
- PostgreSQL Database (Supabase recommended)

## Setup

### 1. Open Project
1. Open project in IntelliJ
2. Click **"Setup SDK"** if prompted → Select Java 17
3. Wait for Maven to download dependencies

### 2. Configure Database
Edit `src/main/resources/application.properties`:

```properties
# Replace with your database credentials
spring.datasource.url=jdbc:postgresql://YOUR-HOST:5432/postgres
spring.datasource.username=postgres
spring.datasource.password=YOUR-PASSWORD
```

**For Supabase:**
1. Go to [supabase.com](https://supabase.com) → Create new project
2. Get connection details from: **Settings → Database → Connection String**
3. Copy URI and update `application.properties`

## Run

### In IntelliJ:
1. Open `src/main/java/backend/MainApplication.java`
2. Click green play button ▶️ next to `main()` method
3. Wait for: `Started MainApplication in X seconds`

### Keyboard shortcuts:
- **Mac:** `⌃ R` or `⇧ F10`
- **Windows/Linux:** `Shift + F10`

## API Endpoints

### Hello Endpoints:
```
GET  http://localhost:8080/
GET  http://localhost:8080/hello
GET  http://localhost:8080/hello?name=YourName
```

### User Endpoints:
```bash
# Get all users
GET http://localhost:8080/api/users

# Get user by ID
GET http://localhost:8080/api/users/{uuid}

# Get user by email
GET http://localhost:8080/api/users/by-email/{email}

# Create user
POST http://localhost:8080/api/users
Content-Type: application/json
{
  "email": "john@example.com",
  "password": "password123"
}
```

### Test with curl:
```bash
# Create user
curl -X POST http://localhost:8080/api/users \
  -H "Content-Type: application/json" \
  -d '{"email":"john@example.com","password":"password123"}'

# Get all users
curl http://localhost:8080/api/users

# Get user by email
curl http://localhost:8080/api/users/by-email/john@example.com
```

## Architecture

```
Request → Controller → Service → Repository → Database
         (REST API)  (Business) (Data Access) (PostgreSQL)
                     (DTOs)      (Entities)
```

- **Entity:** Database table representation (User)
- **Repository:** Database operations (auto-generated by Spring Data JPA)
- **Service:** Business logic and validation
- **Controller:** REST API endpoints
- **DTO:** Data Transfer Objects (request/response)
- **Mapper:** Converts between Entity and DTO (MapStruct)

## Key Technologies

- **Spring Boot 3.2.3** - Framework
- **Spring Data JPA** - Database access
- **Hibernate** - ORM (Object-Relational Mapping)
- **PostgreSQL** - Database
- **MapStruct** - DTO mapping
- **Maven** - Dependency management

## Configuration

### Change Server Port
Edit `application.properties`:
```properties
server.port=8081
```

### Database Settings
```properties
# Auto-create/update tables
spring.jpa.hibernate.ddl-auto=update

# Show SQL queries in console
spring.jpa.show-sql=true
```

## Build

**Maven panel** → **Lifecycle** → Double-click **package**

Creates: `target/backend-0.0.1-SNAPSHOT.jar`

Run JAR:
```bash
java -jar target/backend-0.0.1-SNAPSHOT.jar
```
